
# Projet MVC Complet (Front + Back + CRUD + SQL)

## Fonctionnalités
- CRUD complet sur les événements
- FrontOffice intégré
- BackOffice KaiAdmin intégré
- Base de données SQL fournie
- Structure MVC propre

## Installation
1. Importer la base via `database.sql`
2. Placer le projet dans votre serveur local
3. Ouvrir : http://localhost/projet_final
